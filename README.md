# Copa do Mundo das Vendas — Tênis One

Demonstração do sistema da gincana de vendas da Tênis One.

## Como testar

1. Abra o arquivo `index.html`.
2. Escolha o perfil: **Gerente / Administrador** ou **Vendedor / Acompanhamento**.
3. Digite o PIN demonstrativo `2026`.
4. No perfil gerente, lance vendas, feche rodadas, envie fotos e configure raridades.
5. No perfil vendedor, acompanhe placar, ranking e figurinhas sem alterar dados.
6. Feche a rodada do dia no perfil gerente.
7. Veja o placar das equipes e a artilharia individual.
8. Gere o relatório em PDF no perfil gerente.


## Perfis de acesso

### Gerente / Administrador
O gerente controla o sistema. Ele pode:
- lançar vendas;
- fechar rodadas;
- aplicar bônus;
- enviar fotos dos vendedores;
- definir manualmente quem é Lendária, Ouro, Prata ou Clássica;
- gerar relatório gerencial;
- recriar dados demonstrativos.

### Vendedor / Acompanhamento
O vendedor não cadastra nem altera dados. Ele pode apenas:
- acompanhar o placar;
- acompanhar o ranking;
- ver o álbum e as figurinhas;
- baixar a prévia do álbum.

## Teste gratuito

Esta versão é um teste gratuito por 3 dias a partir da primeira abertura no navegador.

Após o prazo:
- o sistema continua abrindo para visualização;
- novos dados não são salvos;
- ao tentar enviar novos lançamentos, o sistema volta para os dados demonstrativos;
- sugestões e alterações devem ser enviadas para o WhatsApp do Thiago Ventura Valêncio.

## Regras da demonstração

- Time Verde: Isack e Viviane.
- Time Azul: Matheus e Brian.
- A dupla com maior faturamento do dia marca gol.
- Dias 01 a 10: vitória do dia vale 1 gol.
- Dias 11 a 20: vitória do dia vale 2 gols.
- Dias 21 a 30: vitória do dia vale 3 gols.
- Bônus de dezena: +3 gols para a dupla com maior faturamento da dezena nesta demonstração.
- Dupla campeã: R$ 100,00 via PIX para cada integrante + 1 folga premiada.
- Artilheiro individual: R$ 100,00 via PIX.
